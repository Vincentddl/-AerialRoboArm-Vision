# 配置文件说明

本目录保存视觉运行和离线评估使用的相机标定配置。相机、镜头、焦距或分辨率变化后，需要重新标定并更新正式配置。

| 文件 | 中文说明 |
| --- | --- |
| `camera_2p1mm_640x480_fisheye.json` | 当前正式使用的 2.1 mm 鱼眼镜头标定参数，分辨率为 `640x480`。运行识别、方向角映射和离线评估时优先使用。 |
| `calibration.example.json` | 标定配置示例文件，用于说明配置格式或给旧脚本做参考。 |

## 注意事项

- `camera_2p1mm_640x480_fisheye.json` 与当前模型和方向角评估强相关，不要随意替换。
- 如果更换摄像头或镜头，需要重新运行标定流程，并同步更新 README 中的说明。
