# 数据集目录说明

本目录保存视觉训练和轨迹评估数据。当前只上传了协作者优化模型所需的关键数据，其余历史数据仍按 `.gitignore` 保留在本地。

## 已上传数据

| 路径 | 中文说明 |
| --- | --- |
| `foam_board_2p1mm/v7/` | 当前 YOLO 检测模型的主要训练/验证数据集，包含图片、YOLO 标签、数据集配置和清单。 |
| `foam_board_2p1mm_zaxis/` | Z-axis 轨迹录像、时间戳、V6/V7 检测结果和人工筛选辅助图，用于方向角预测和轨迹评估。 |

## 未上传数据

历史版本数据集、完整原始采集和大量中间结果默认不上传。原因是文件数量和体积较大，不适合全部放进普通 Git 协作流程。

## 协作建议

- 只优化 YOLO 检测模型时，优先使用 `foam_board_2p1mm/v7/`。
- 优化方向角预测或轨迹评估时，同时使用 `foam_board_2p1mm_zaxis/` 和 `outputs/` 中的相关 JSON/JSONL。
- 数据集图片和视频通过 Git LFS 管理。克隆后如果文件不能正常打开，请先运行 `git lfs pull`。
