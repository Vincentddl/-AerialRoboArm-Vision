# Tracking 模块说明

本目录保存运行时轨迹预测、坐标映射和方向角预测逻辑。它是视觉检测结果进入机械臂目标输出之前的核心处理层。

| 文件 | 中文说明 |
| --- | --- |
| `trajectory.py` | 历史二维像素轨迹模块，包含像素位置、速度、加速度 Kalman 预测和相机标定映射。当前保留为基线与兼容实现。 |
| `bearing.py` | 当前方向角实验模块，负责鱼眼去畸变后的方向角映射、角度误差计算、保持角度预测和鲁棒多项式预测。 |

## 使用建议

- 若目标是优化未来抓取点预测，优先阅读 `bearing.py`。
- 若目标是理解旧版像素轨迹输出格式，阅读 `trajectory.py`。
- 修改角度计算后必须运行 `tests/test_bearing.py`。
